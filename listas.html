Para crear una lista desordenada empiezo utilizando la etiqueta "ul" y para una lista ordenada la etiqueta "ol", dentro de cada lista para cada elemento de la lista utilizo la etiqueta "li".La lista ordenada muestra ítems con números o letras en orden secuencial. Se usa cuando el orden de los elementos es importante, en cambio en las listas desordenadas se muestran ítems con viñetas (puntos) y no tiene un orden específico. Se usa cuando el orden de los elementos no es importante.
